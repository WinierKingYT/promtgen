Gemini API Yapılandırması

Fikirlerinizin yapay zeka tarafından gerçek zamanlı analiz edilmesi için bir AI Sağlayıcı seçin ve API anahtarınızı girin.

API anahtarınız PromtGen'e ait bir sunucuda saklanmaz. İstek sırasında yalnızca seçtiğiniz AI sağlayıcısına gönderilir. Anahtar tarayıcınızın localStorage belleğinde şifrelenmeden saklanır.

API Anahtarı Olmadan Çalışma (Çevrimdışı Mod)

Eğer bir API anahtarınız yoksa, sistem anahtar kelime eşlemeli Çevrimdışı Şablon Motorunu (yerel bir yapay zeka modeli yerine önceden tanımlanmış statik mimari şablon fall-back sistemini) kullanarak yüksek kaliteli prompt dizilimleri üretecektir.
